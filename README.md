A cute yet powerful all-in-one study productivity web app built using **HTML, CSS, and Vanilla JavaScript**.

StudyFlow helps students manage subjects, track progress, use a Pomodoro timer, and revise with interactive flashcards — all in one aesthetic dashboard.

---

## ✨ Features

- 📊 **Dashboard** – Study stats, weekly tracking, achievements  
- 📚 **Subject Management** – Priority, deadlines, notes, progress bars  
- ⏱️ **Pomodoro Timer** – Focus/Break modes, animated ring, sound effects  
- 🃏 **Flashcards** – Custom decks, flip animation, shuffle, accuracy tracking  
- 📈 **Progress Page** – Weekly chart + accuracy doughnut (Chart.js)  
- 💾 **Data Persistence** – Auto-save using LocalStorage  
- 🎨 **UI Enhancements** – Dark/Light theme, toast notifications, confetti  
- ⌨️ **Keyboard Shortcuts** – Faster navigation  

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES6)  
- Chart.js  
- Web Audio API  
- LocalStorage API  

Fully client-side. No backend.
